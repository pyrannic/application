## Getting Started

#### Create an environment

To begin, set up a new environment using your preferred method.
If you need it, the [FastAPI documentation](https://fastapi.tiangolo.com/virtual-environments/#create-a-virtual-environment) provides an excellent guide on this process.


#### Install Dependencies

Once your environment is set up, you can install the project dependencies using your package manager of choice:

```bash
pdm install
# or
uv sync
# or
poetry install
```

#### Run your App

Finally, run the FastAPI development server:

```bash
fastapi dev
```

Open [http://localhost:8000/docs](http://localhost:8000/docs) with your browser to see and execute the example endpoint provided by the skeleton project.

You can experiment with changing this endpoint by editing `app/http/routers/default.py`.
